# 栈

## 定义

栈是一种只能在一端进行的插入或删除操作的线性表。

* **栈顶：**表中允许进行插入和删除的一端
* **栈底：**栈顶对应的另一端被称为栈底
* **入栈\(Push\)：**栈的插入称为入栈
* **出栈\(Pop\)：**栈的删除称为出栈

## 特点

栈最明显的特点就是**先进后出（FILO）。**
