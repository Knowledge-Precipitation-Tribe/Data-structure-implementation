# 队列

## 定义

队列是操作受到限制的线性表，表现在只能在表的一端进行插入，在另一段进行删除。

* **队尾\(Rear/Back\)：**可进行插入的一端称为队尾
* **队头\(Front\)：**可进行删除的一端称为队头
* **入队\(enQueue\)：**向队列中插入新的元素称为进队
* **出队\(deQueue\)：**从队列中删除元素称为出队

## 特点

队列最明显的特点就是**先进先出（FIFO）。**
